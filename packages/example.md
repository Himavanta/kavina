# example

内置示例部件包。适合上手体验和参照开发。

## 部件列表

| 部件 | 说明 |
| --- | --- |
| `text` | 文本显示 |
| `button` | 按钮 |
| `input` | 输入框 |
| `form` | 表单容器 |
| `layout` | 布局容器 |
| `tabs` | 选项卡 |
| `array` | 数组容器，支持动态增删子节点 |
| `group` | 编组容器，拖入子节点后自动包裹 |
| `stencil` | 模板部件，支持表达式渲染 |
| `frame` | 框架部件 |
| `demo` | 演示部件 |

## 使用

```ts
import { example } from 'kavina/packages/example'

const widgets = [example]
```

## 前置

无。不依赖外部 UI 库。
